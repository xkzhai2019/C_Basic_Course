##### 6. 如何定义变量

数据类型 变量名  = 要赋的值

等价于

int i = 2;  ------------------------ int i; i=2;
int i,j;  ----------------------------int i; int j;

int i = 3, j = 5; -------------------- int i; int j; i = 3; i = 5;

##### 8. 常量在C语言中如何表示？
