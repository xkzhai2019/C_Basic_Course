#### C语言概述

##### 1. C语言的起源与特点
* C的特点：代码量小、速度快、功能强大
* 缺点：危险性高、开发周期长、可移植性不强
  
##### 2. C的应用领域

系统软件开发：
* 操作系统：
windows: C,C++
macOS: C
Linux: C
  
* 驱动程序：主板驱动、显卡驱动、摄像头驱动

应用软件开发：WPS、图形图像多媒体

嵌入式软件开发：智能手机、掌上电脑

游戏开发：2D\3D游戏

* 为什要学C语言（Java,C++）
* 有助于理解面向对象
* 学C语言就是在学C++
